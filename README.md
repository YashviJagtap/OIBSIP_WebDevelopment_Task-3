# OIBSIP Web Development Task 3 - Temperature Converter

## Objective
The objective of this project is to build a temperature converter that converts values between Celsius, Fahrenheit, and Kelvin using HTML, CSS, and JavaScript.

## Tools & Technologies Used
- HTML
- CSS
- JavaScript
- Visual Studio Code
- Live Server

## Steps Performed
1. Created an input field for entering temperature.
2. Added a dropdown menu to select the temperature unit.
3. Implemented a Convert button to trigger the conversion.
4. Used JavaScript to calculate and display the converted temperature.
5. Added styling using CSS for better user interface.

## Outcome
A functional temperature converter that allows users to convert temperature values between Celsius, Fahrenheit, and Kelvin.
